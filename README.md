# Etch-A-Sketch #
The classic drawing toy, digitized. Built in vanilla HTML, CSS, and JS.

## How to Use
1. Use the slider to adjust the grid size (1-100)
2. Select your desired color using the color picker
3. Hover your mouse over the canvas to draw
4. Click the "Clear" button to erase and start over
